## Sideloading (APK)

If you prefer to install manually or cannot access the Play Store:

1. Download the latest `app-release.apk` from the [GitHub Releases](https://github.com/EarMaster/AutoSugar/releases) page.
2. On your Android phone, go to **Settings → Apps → Special app access → Install unknown apps** and allow installations from your browser or file manager.
3. Open the downloaded APK and follow the installation prompts.
4. To use AutoSugar with Android Auto, you also need to enable **Unknown sources** in the Android Auto developer settings:
   - Open Android Auto → tap the version number in the footer 10 times to unlock developer settings.
   - In **Developer settings**, enable **Unknown sources**.

> **Note:** Sideloaded apps do not receive automatic updates. Check the [releases page](https://github.com/EarMaster/AutoSugar/releases) periodically for new versions.
