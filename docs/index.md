# AutoSugar

Blood glucose monitoring for Android Auto — see your Nightscout CGM data at a glance, right on your car's display.

[![Get it on Google Play](https://play.google.com/intl/en_us/badges/static/images/badges/en_badge_web_generic.png)](https://play.google.com/store/apps/details?id=de.autosugar){:height="60px"}

## Features

- Monitor one or multiple Nightscout sources (e.g. yourself and your children)
- Current glucose value, trend arrow, and delta
- Configurable glucose alerts
- English and German UI

## Source code

AutoSugar is open source. Find the code and report issues on [GitHub](https://github.com/EarMaster/AutoSugar).
